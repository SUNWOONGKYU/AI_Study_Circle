# WAAT 문자/알림톡 운영 방안

작성일: 2026-05-10
상태: **준비 단계 — 추후 통합 예정**

---

## 1. 결정 사항

| 항목 | 결정 |
|---|---|
| 솔루션 | **알리고 (Aligo)** — https://smartsms.aligo.in/ |
| 메시지 종류 | 단문 SMS + 카카오 알림톡 (2채널 통합) |
| 발송 빈도 | 수시 (모임 안내, 리마인더, 공지) |
| 예상 비용 | 월 1,500~3,000원 (회원 50명 기준) |

### 가격 (2026년 5월 기준)
- SMS 단문: 8.4원/건
- 알림톡: 6.5~8원/건
- 가입비/월정액: 무료 (충전식)

## 2. 운영 시나리오

### 2-1. 자동 발송 (시스템 트리거)

| 시점 | 메시지 종류 | 내용 |
|---|---|---|
| 가입 직후 | 알림톡 | "WAAT 가입을 환영합니다. 모임 일정은..." |
| 참석 신청 직후 | 알림톡 | "{N회 모임 / 햇살 슬롯} 신청이 완료되었습니다." |
| 모임 D-3 | 알림톡 | 장소·시간 리마인더 |
| 모임 D-1 | 알림톡 | 최종 리마인더 |
| 모임 D-day 오전 | SMS | 당일 안내 (간단) |
| 신청 취소 시 | 알림톡 | 취소 확인 |

### 2-2. 수동 발송 (관리자)

- admin.html에 "전체 회원 공지 발송" 버튼
- 특정 회차 참석자만 골라 발송
- 메시지 미리보기 + 예상 비용 표시 후 확인

### 2-3. 발송 우선순위

```
[1차] 알림톡 시도 (카카오톡 사용자)
       ↓ 실패/미수신
[2차] SMS 자동 전환 (Aligo 자체 기능 활용)
```

## 4. 개발 작업 항목

### 4-1. DB 스키마 추가
```sql
-- 메시지 발송 로그
CREATE TABLE message_logs (
    id BIGSERIAL PRIMARY KEY,
    member_id UUID REFERENCES members(id),
    channel TEXT NOT NULL,          -- 'alimtalk' | 'sms'
    template_code TEXT,              -- 알림톡 템플릿 식별자
    phone TEXT NOT NULL,
    body TEXT NOT NULL,
    status TEXT NOT NULL,            -- 'queued' | 'sent' | 'failed'
    aligo_msg_id TEXT,               -- 알리고 응답 ID
    error_msg TEXT,
    cost_won INTEGER,                -- 발송 비용
    sent_at TIMESTAMPTZ DEFAULT NOW()
);

-- 회원 동의 플래그
ALTER TABLE members
    ADD COLUMN sms_optin BOOLEAN DEFAULT true,
    ADD COLUMN alimtalk_optin BOOLEAN DEFAULT true;
```

### 4-2. Supabase Edge Function: `send-notification`
- 알리고 API 호출 (알림톡 우선, SMS fallback)
- 템플릿 코드 + 변수(이름, 회차, 날짜 등) 치환
- 발송 결과를 `message_logs`에 기록
- 회원 opt-out 체크

### 4-3. 자동 트리거
- DB 트리거 또는 Edge Function:
  - 신규 가입 → 환영 알림톡
  - 신청/취소 → 확인 알림톡
- Cron (Supabase Scheduled Function):
  - 매일 오전 9시 → D-3, D-1 리마인더 자동 발송

### 4-4. 어드민 UI
- `admin.html` → "메시지 발송" 탭 신설
- 수신자 선택 (전체 회원 / N회 참석자 / 개별 선택)
- 메시지 작성 + 템플릿 미리보기
- 예상 비용 계산 + 발송 확인 모달
- 발송 이력 조회 (성공/실패/비용)

## 5. 알림톡 템플릿 초안 (카카오 검수용)

### T1. 가입 환영
```
[WAAT] {이름}님, 가입을 환영합니다!

매주 수요일 오후 AI 관련 자유로운 수다 모임입니다.
다음 모임: {다음모임날짜}
장소: {장소}

언제든 편하게 오세요.
```

### T2. 신청 확인
```
[WAAT] 신청이 완료되었습니다.

▷ 회차: 제{N}회 모임
▷ 날짜: {모임날짜} (수)
▷ 시간대: {슬롯이름} ({시작}~{종료})
▷ 장소: {장소}

곧 뵙겠습니다!
```

### T3. D-1 리마인더
```
[WAAT] 내일 모임 안내

▷ 일시: {모임날짜} {요일}
▷ 시간대: {슬롯이름} ({시작}~{종료})
▷ 장소: {장소}
▷ 주소: {주소}

지도: {지도URL}
```

### T4. 신청 취소 확인
```
[WAAT] 신청 취소가 처리되었습니다.

▷ 취소된 모임: 제{N}회 ({날짜})
다음에 또 만나요!
```

### T5. 일반 공지 (관리자 수동)
```
[WAAT] 공지

{본문}
```
