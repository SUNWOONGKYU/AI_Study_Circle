## 1. 사전 준비 (최초 1회만)

### 1-1. GitHub 계정 만들기

1. https://github.com 에 접속
2. **Sign up** 클릭 → 이메일, 비밀번호, 사용자명 입력하여 가입
3. 이메일 인증 완료

### 1-2. 저장소 권한 받기

- 관리자(wksun999)에게 **본인의 GitHub 사용자명**을 알려주세요
- 관리자가 저장소에 **Collaborator(협력자)**로 추가해줍니다
- GitHub에서 초대 이메일이 오면 **Accept** 클릭

### 1-3. Git 설치하기

1. https://git-scm.com/download/win 에 접속
2. **Download for Windows** 클릭하여 설치 파일 다운로드
3. 설치 진행 (모든 옵션 기본값 그대로 **Next** 클릭)
4. 설치 완료 확인:
   - 바탕화면이나 폴더에서 **마우스 오른쪽 클릭**
   - **"Open Git Bash"** 또는 **"Git Bash Here"** 메뉴가 보이면 성공

### 1-4. Git 사용자 정보 설정 (최초 1회)

Git Bash를 열고 아래 두 줄을 입력합니다 (본인 정보로 수정):

```bash
git config --global user.name "본인이름"
git config --global user.email "lsonic.lee@gmail.com"
```

### 1-5. Visual Studio Code 설치 (권장)

코드를 편하게 수정하기 위한 무료 편집기입니다.

1. https://code.visualstudio.com 에 접속
2. **Download for Windows** 클릭하여 설치
3. 설치 시 **"Add to PATH"** 체크 (기본 체크되어 있음)

## 2. 사이트 코드 내려받기 (클론)

### 2-1. 작업 폴더 만들기

1. 원하는 위치에 폴더를 만듭니다 (예: `C:\Projects`)
2. 해당 폴더에서 **마우스 오른쪽 클릭** → **"Git Bash Here"** 클릭

### 2-2. 코드 내려받기

Git Bash 창에 아래 명령어를 복사 붙여넣기 후 Enter:

```bash
git clone https://github.com/SUNWOONGKYU/AI_Study_Circle.git
```

완료되면 `AI_Study_Circle` 폴더가 생깁니다.

### 2-3. 폴더 열기

```bash
cd AI_Study_Circle
```

또는 Visual Studio Code에서:
- **파일** → **폴더 열기** → `AI_Study_Circle` 폴더 선택

## 3. 코드 수정하기

### 파일 구조 안내

```
AI_Study_Circle/
├── index.html          ← 메인 페이지 (가장 자주 수정)
├── admin.html          ← 관리자 페이지

│   └── animations.js   ← 애니메이션 효과

    └── schema.sql      ← DB 테이블 구조 (참고용)
```

### 수정 방법

2. 수정할 텍스트를 찾습니다 (**Ctrl + F**로 검색)
3. 수정 후 **Ctrl + S**로 저장합니다

다른 사람이 수정했을 수 있으므로, **수정 전에 항상** 이 명령어를 실행하세요:

```bash

```

## 4. 수정한 내용 반영하기 (커밋 & 푸시)

수정을 저장했으면, 아래 3단계로 사이트에 반영합니다.

### 4-1. 변경 파일 확인

```bash
git status
```

수정된 파일이 빨간색으로 표시됩니다.

### 4-2. 변경 파일 추가 + 커밋 (기록 남기기)

```bash
git add .
git commit -m "여기에 무엇을 수정했는지 적기"
```

예시:
```bash
git add .
git commit -m "히어로 문구 수정"
```

### 4-3. 사이트에 반영 (푸시)

```bash
git push
```

> GitHub 로그인 창이 뜨면 로그인하세요 (최초 1회).

### 4-4. 확인

- 1~2분 후 https://sunwoongkyu.github.io/AI_Study_Circle/ 에 접속하여 반영 확인
- 안 보이면 **Ctrl + Shift + R** (강력 새로고침)

## 6. admin 페이지에서 할 수 있는 것

코드 수정 없이 admin 페이지에서 할 수 있는 작업들:

| 기능 | 경로 |
|------|------|
| 모임 등록/수정/비활성화 | admin 페이지 > 모임 관리 탭 |
| 장소 등록/수정/삭제 | admin 페이지 > 장소 관리 탭 |
| 멤버 목록 조회/검색 | admin 페이지 > 멤버 관리 탭 |
| 모임 참여자 명단 조회 | admin 페이지 > 모임 관리 > 보기 버튼 |

**admin 페이지 접속 방법:**
1. 메인 사이트에서 관리자 계정으로 로그인
2. 우측 상단 이름 클릭 → **"관리자 페이지"** 클릭
3. 또는 직접 접속: `https://sunwoongkyu.github.io/AI_Study_Circle/admin.html`

> 모임 일정, 장소 정보, 참여자 관리는 코드를 수정할 필요 없이 admin 페이지에서 가능합니다!

## 7. 문제가 생겼을 때

### "충돌(conflict)"이 발생했을 때

두 사람이 같은 파일을 동시에 수정하면 충돌이 발생할 수 있습니다.

```bash
git pull
```

했을 때 충돌이 나면:

1. 당황하지 마세요
2. 아래 명령어로 내 수정을 취소하고 원래 상태로 되돌릴 수 있습니다:

```bash
git merge --abort
git stash
git pull
```

그 후 다시 수정하면 됩니다.

### 내 수정을 전부 취소하고 싶을 때

```bash
git checkout .
```

이 명령어는 수정한 내용을 전부 되돌립니다 (커밋 전에만 가능).

### push가 거부될 때

다른 사람이 먼저 push한 경우입니다:

```bash
git pull
```

한 후 다시:

```bash
git push
```

## 요약: 수정 작업 흐름

```
1. Git Bash 열기 (프로젝트 폴더에서)
2. git pull                          ← 최신 코드 받기
3. 파일 수정 (VS Code 등에서)
4. git add .                         ← 변경 파일 추가
5. git commit -m "수정 내용 설명"     ← 기록 남기기
6. git push                          ← 사이트에 반영
```

이 6단계만 기억하면 됩니다!
